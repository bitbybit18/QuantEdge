# QuantEdge# 📈 QuantEdge — AI-Powered Financial Intelligence Dashboard

> A full-stack fintech application that combines real-time market data,
> machine learning predictions, sentiment analysis, and trading signals
> into a professional-grade investment dashboard.

---

## 🚀 Features

- 📊 **Real-time Stock Data** — Live market prices and OHLCV charts
- 🤖 **AI Price Prediction** — ML models (Linear Regression → LSTM)
- 📰 **Sentiment Analysis** — News sentiment using FinBERT
- 📉 **Trading Signals** — SMA, EMA, RSI-based buy/sell signals
- ⚠️ **Risk Analysis** — Volatility scoring and risk metrics
- ⛓️ **Blockchain Logging** — Immutable trade signal logging (optional)

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React, Vite, Tailwind CSS, Recharts |
| Backend | Python, FastAPI |
| Machine Learning | scikit-learn, TensorFlow/Keras, XGBoost |
| NLP | FinBERT (HuggingFace Transformers) |
| Data | yfinance, Alpha Vantage API |
| Deployment | Vercel (frontend), Render (backend) |
